## ✨ 功能特色

### 📝 測驗模式
- 選擇題（四選一）
- **循序作答**或**隨機出題**
- 點選後立即顯示正確/錯誤
- 每題附**詳細中文解析**（含時序、機制、鑑別診斷）
- 即時正確率顯示
- 結束後顯示成績 + **全題答題回顧**

### 🃏 閃卡記憶
- 點擊卡片翻面查看答案
- 上下張導航
- 隨機排序
- 進度圓點指示
- 完整解析顯示

### 🎨 設計細節
- 每章使用**獨立主題色**，視覺上易區分章節
- **深色模式**護眼設計
- 響應式版面，**手機、平板、電腦皆適用**
- 無需安裝、無需登入、無廣告

---

## 💡 學習建議

### 三輪複習法
1. **第一輪**：循序作答，熟悉題型 → 看詳解理解概念
2. **第二輪**：用閃卡快速記憶，遮住答案自我測試
3. **第三輪**：隨機出題，模擬實際考試

### 重點章節分配
- **生殖系統（54 題）**最大章，需多花時間
- **造血淋巴（39 題）**、**消化（37 題）**也是大章
- **神經系統（15 題）**題數最少，但臨床重要

---

## 🛠️ 技術說明

- **純前端**：HTML + CSS + JavaScript（無需後端、無需資料庫）
- **單檔架構**：每章一個 HTML 檔，可離線使用
- **無外部依賴**：不需要 npm、無 framework，瀏覽器直接打開即用
- **無追蹤**：不收集任何使用者資料
